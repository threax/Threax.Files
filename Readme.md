# Threax.Files
File browser app.

## Changing Icon and Site Name
You can change the icon and the Site Name by adding the following to your config
```
  "DisplayConfig": {
    "SiteName": "Music",
    "Favicon": "~/icons/favicon_music.ico",
    "Icon": {
      "src": "~/icons/icon_music.png",
      "sizes": "512x512"
    }
  }
```
